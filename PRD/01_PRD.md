# comad-stock — PRD

## 1. 개요

**comad-stock**은 DART 공시와 GraphRAG 기반 엔티티 관계 추론을 활용한, 한국어 우선 OSS 주식 리서치 터미널이다. 한국 리테일 투자자에게 Fey/Seeking Alpha 수준의 진지한 분석 도구를 무료로 제공하는 것을 목표로 한다.

## 2. 해결하는 문제

한국 리테일 투자자 중 DIY fundamental 성향 사용자는 DART 공시를 직접 읽고 네이버 증권·증권사 앱을 전전한다. 공시 해석, 기업 간 관계 추론(공급망·경쟁사), 체계적 DD 메모 작성은 모두 수작업이다. 영어권에는 Fey·Seeking Alpha 같은 도구가 있으나 한국 종목·한국어를 지원하지 않는다.

## 3. 타겟 유저

- **1차:** 토스증권·삼성·미래에셋 앱 사용자 중 DART 공시를 읽는 DIY 투자자
- **2차:** 한국 주식 추적 해외 투자자(영문 자동 번역), 소형주·IPO 리서처

## 4. 북극성 지표 (두 개)

1. **기술적 완성도** — 단순 요약이 아닌 GraphRAG 기반 관계 추론. "HBM 공급망 중 이상 공시" 같은 다단계 질의가 실제로 작동.
2. **Fey급 UI/UX** — 광고 0, 정보 밀도와 심미성 양립. 네이버 증권 수준을 두 세대 뛰어넘음.

## 5. MVP 기능 (Phase 1)

1. **종목 대시보드** — 시세(15분 지연), 기본 재무, 공시 타임라인, 관련 뉴스를 한 화면에. KOSPI 200 전체 커버.
2. **DART 공시 AI 요약 + 이상징후 감지** — 자동 수집·한국어 요약·이상 신호 플래그(severity low/med/high).
3. **AI DD 메모 자동 생성** — 종목 입력 → bull/bear 한국어 메모(버전 히스토리, 출처 각주 포함).
4. **GraphRAG 자연어 Q&A** — 기업 엔티티 그래프(경쟁사·공급망·인물) 위 자연어 질의. Cypher 자동 생성 + pgvector 하이브리드 검색.

## 6. 플랫폼

- **데스크탑 웹 + 모바일 웹 반응형** — 동등 지원. Next.js 14 App Router + RSC.

## 7. 인증

- **MVP:** 이메일+비밀번호
- **Phase 2:** 메일 서버 구축 후 매직링크로 전환

## 11. 핵심 리스크

2. **LLM 비용** — DD 메모 1건 ~$0.10 → BYOK로 해결
3. **규제** — 투자자문업 아님 명시, 매수·매도 추천 프롬프트 금지
4. **Fey급 UX 솔로 리스크** — 디자인 시스템 초기 투자 필수
