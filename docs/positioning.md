# Positioning — 확정본

**결정일:** 2026-04-14
**상태:** 확정 (research/market-landscape.md의 A/B/C/D 가설에서 재구성됨)

## 한 줄 포지션

> 관심종목의 중요한 DART 공시를 놓치지 않게 하는 AI 공시 레이더. 진지한 한국 개인투자자 우선, GraphRAG와 DD 메모는 심화 리서치 레이어.

## 북극성 (두 개)

1. **리테일 실효성** — 관심종목 기준으로 오늘 봐야 할 공시, 위험 신호, 확인할 체크포인트를 먼저 제시.
2. **기술적 완성도** — 공시 요약 수준을 넘어, 기업 간 관계·공급망·이상징후를 GraphRAG로 추론. 단순 LLM 래핑 제품과 본질적으로 다름.

두 축을 동시에 최고 수준으로 끌어올리는 것이 이 프로젝트의 존재 이유. 하나라도 평범하면 또 하나의 요약기/또 하나의 대시보드가 됨.

## 타겟 유저

**1차:** 한국 리테일 투자자 중 DIY fundamental 성향. 특히 DART 공시, 유상증자/CB, 최대주주 변경, 감사의견, 지배구조를 직접 챙기는 사람. 거래는 토스증권/삼성/미래에셋 앱에서 하지만, 리서치와 공시 해석은 별도 도구가 필요한 사용자.

**2차:** 국내 소형주·IPO 리서처, 한국 주식을 추적하는 해외 투자자(번역 자동화).

**장기 옵션:** 기관/운용사 PoC. 단, 1차 제품 판단 기준은 기관 컴플라이언스가 아니라 개인 투자자의 매일 쓰는 공시 워크플로다.

## 타겟이 아닌 것

- 데이트레이더 (차트·호가창 중심) — TradingView가 이미 지배
- 패시브 장기 투자자 (인덱스 ETF만) — 분석 불필요
- 기관 투자자 — 단기 1차 타겟 아님. 컴플라이언스·계약·SLA 요구는 별도 로드맵에서 다룸

## 4개 차별점 (자세히)

### 1. 관심종목 공시 레이더
- 사용자가 등록한 종목의 최신 공시와 high/medium 이벤트를 먼저 보여준다.
- reviewed / dismissed / escalated 상태를 저장해 "봤는지"를 남긴다.
- 알림은 Ask보다 우선순위가 높다. 개인 투자자는 질문을 매일 던지기보다 신호를 먼저 받고 싶어한다.

### 2. DART 공시 AI 분석
- 반기·사업보고서, 주요사항보고서, 지분변동 공시, 투자판단관련 주요경영사항 등을 자동 수집·요약.
- 단순 요약이 아닌 **이상징후 감지**: 전분기 대비 비정상 변동, CEO/대주주 움직임, 감사인 변경 플래그.
- 한국어 원문 + 영문 번역 동시 제공.

### 3. AI DD 메모 자동 생성
- 종목 입력 → 에이전트가 한국어 DD 메모 생성 (bull/bear, 경쟁 구도, 공시 요약, 실적 델타, 내부자 거래, 리스크).
- 출처 각주 포함 (환각 방지).
- 포트폴리오 단위 일괄 재생성 (주간 브리핑 용도).

### 4. GraphRAG 엔티티 그래프
- 기업-자회사-경쟁사-공급망-주요 인물을 Neo4j에 구축. comad-brain 그대로 재활용.
- 자연어 Q&A: "내 워치리스트 중 HBM 공급망 관련 최근 이상 공시"
- 관계 추론: 삼성전자 공시가 SK하이닉스에 미치는 2차 영향 자동 표시.
