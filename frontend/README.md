# comad-stock frontend

Next.js 16 (App Router, React 19) + Tailwind 4 + Recharts 3.

**디자인 시스템:** `../docs/DESIGN.md` — Fey급 UI 목표, editorial + dark-first.

## 개발

```bash
npm install
npm run dev   # http://localhost:3333
```

## 랜딩 페이지 검증

`src/app/page.tsx`가 Display 타이포그래피(EB Garamond) + Pretendard 본문 + mono 메타 조합으로
디자인 시스템을 첫 검증한다. 스크린샷 확인 후 대시보드 컴포넌트 구현 진입.
