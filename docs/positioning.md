# Positioning — 확정본

**결정일:** 2026-04-14
**상태:** 확정 (research/market-landscape.md의 A/B/C/D 가설에서 재구성됨)

## 한 줄 포지션

> DART-native AI 주식 리서치 터미널. Fey의 UI/UX 수준 + GraphRAG 기반 기술 완성도, 한국어 우선, OSS.

## 북극성 (두 개)

1. **기술적 완성도** — 공시 요약 수준을 넘어, 기업 간 관계·공급망·이상징후를 GraphRAG로 추론. 단순 LLM 래핑 제품과 본질적으로 다름.
2. **Fey급 UI/UX** — 광고 0, 정보 밀도와 심미성 양립. 한국 금융 UX의 전반적 낙후(네이버·증권사 앱)를 두 세대 건너뜀.

두 축을 동시에 최고 수준으로 끌어올리는 것이 이 프로젝트의 존재 이유. 하나라도 평범하면 또 하나의 요약기/또 하나의 대시보드가 됨.

## 타겟 유저

**1차:** 한국 리테일 투자자 중 DIY fundamental 성향 (토스증권/삼성/미래에셋 앱으로 거래하지만 분석 도구는 부족한 사람). Seeking Alpha/Fey를 영어 장벽이나 가격으로 못 쓰던 사용자.

**2차:** 한국 주식을 추적하는 해외 투자자 (번역 자동화). 국내 소형주·IPO 리서처.

## 타겟이 아닌 것

- 데이트레이더 (차트·호가창 중심) — TradingView가 이미 지배
- 패시브 장기 투자자 (인덱스 ETF만) — 분석 불필요
- 기관 투자자 — 컴플라이언스·계약 장벽

## 3개 차별점 (자세히)

### 1. DART 공시 AI 분석
- 반기·사업보고서, 주요사항보고서, 지분변동 공시, 투자판단관련 주요경영사항 등을 자동 수집·요약.
- 단순 요약이 아닌 **이상징후 감지**: 전분기 대비 비정상 변동, CEO/대주주 움직임, 감사인 변경 플래그.
- 한국어 원문 + 영문 번역 동시 제공.

### 2. GraphRAG 엔티티 그래프
- 기업-자회사-경쟁사-공급망-주요 인물을 Neo4j에 구축. comad-brain 그대로 재활용.
- 자연어 Q&A: "내 워치리스트 중 HBM 공급망 관련 최근 이상 공시"
- 관계 추론: 삼성전자 공시가 SK하이닉스에 미치는 2차 영향 자동 표시.

### 3. AI DD 메모 자동 생성
- 종목 입력 → 에이전트가 한국어 DD 메모 생성 (bull/bear, 경쟁 구도, 공시 요약, 실적 델타, 내부자 거래, 리스크).
- 출처 각주 포함 (환각 방지).
- 포트폴리오 단위 일괄 재생성 (주간 브리핑 용도).

## 경쟁 구도

| 서비스 | 강점 | 약점 (comad-stock이 이기는 지점) |
|---|---|---|
| 네이버 증권 | 트래픽·뉴스 | 광고, 분석 도구 없음, UX 낡음 |
| 토스증권 인사이트 | UX 좋음, 토스 생태계 | 단편적 콘텐츠, DD 불가, GraphRAG 없음 |
| 증권플러스 | 시세 편의 | 분석 없음 |
| 키움·삼성 등 브로커 앱 | 거래 | 분석 도구는 광고·편향 |
| 퀀트킹/아이투자 | 정량 스크리너 | UX 낡음, 공시 AI 없음 |
| Fey / Seeking Alpha | 글로벌 리서치 수준 | 한국 종목·DART 커버 없음, 영어 |

**공백:** "Fey 수준 UX + 한국 공시 깊이"는 어느 경쟁자도 하고 있지 않음.
